# 3.2.4. Usability & Accessibility

- **Consistent design system.** The interface follows a **Material Design 3**
  token system for colors, typography, and spacing, applied consistently across
  views.
- **Internationalization.** The UI is fully internationalized across **six
  locales**, with the user's preferred locale persisted in `localStorage`.
- **Responsive layouts.** Layouts adapt from mobile to desktop, including
  responsive navigation and feed/card layouts.
- **Light & dark themes.** The application supports light and dark palettes.
- **Clear feedback.** Actions surface clear feedback through notifications and
  status indicators, so users always understand the result of an interaction.
