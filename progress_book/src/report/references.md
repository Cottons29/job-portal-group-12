# 10.1. References

The project was built on the following frameworks, libraries, and standards:

### Frameworks & libraries
- [NestJS](https://nestjs.com/) — backend application framework.
- [Vue 3](https://vuejs.org/) — frontend framework.
- [Vite](https://vitejs.dev/) — frontend build tool and dev server.
- [Pinia](https://pinia.vuejs.org/) — Vue state management.
- [TypeORM](https://typeorm.io/) — ORM for PostgreSQL access.
- [PostgreSQL](https://www.postgresql.org/) — relational database.
- [vue-i18n](https://vue-i18n.intlify.dev/) — internationalization.

### Standards & specifications
- [WebAuthn / Passkeys](https://www.w3.org/TR/webauthn-2/) — passwordless
  authentication.
- [Material Design 3](https://m3.material.io/) — design system.

### Tooling & infrastructure
- [Docker](https://www.docker.com/) and Docker Compose — containerization.
- [GitHub Actions](https://docs.github.com/actions) — CI/CD.
- [GitHub Container Registry (GHCR)](https://docs.github.com/packages) — image
  registry.
- [Rust](https://www.rust-lang.org/) — the standalone file service.
- [mdBook](https://rust-lang.github.io/mdBook/) — this documentation book.
