# 2.2. Project Objectives

The project set out to deliver, within a single semester, a production-grade
platform that achieves the following objectives:

1. **Unify discovery.** Provide a single searchable, filterable marketplace for
   student part-time jobs, replacing scattered, informal channels.
2. **Establish trust.** Support verified student identities (including student ID
   card upload for admin verification) and distinct, role-aware employer
   profiles.
3. **Improve relevance.** Implement a skill-matching recommendation engine that
   scores how well a student's profile fits a job's requirements.
4. **Make the process transparent.** Model an explicit recruitment pipeline so
   applicants can see their status move from *Submitted* through *Viewed*,
   *Interview*, *Offer*, and *Result*.
5. **Keep communication in-platform.** Offer direct messaging and real-time
   notifications so context is never lost.
6. **Be secure by default.** Use session and passkey (WebAuthn) authentication,
   encrypt sensitive auth payloads, and provide secure account recovery.
7. **Be accessible to everyone.** Internationalize the interface across six
   locales and follow a consistent Material Design 3 system.
8. **Ship reliably.** Automate building, testing, and deployment of every
   component through a CI/CD pipeline.
